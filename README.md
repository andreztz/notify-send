# notify-send

![](header.png)

## Installation

```sh
    pip install notify-send
```

## Usage example

```python
    from notify import Notification
    Notification('title', 'body message')
```

## Development setup

```sh
    virtualenv venv
    source venv/bin/activate
    python setup.py develop
```
